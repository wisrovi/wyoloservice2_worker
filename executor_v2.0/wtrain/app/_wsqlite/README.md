# wsqlite Package

Main library package providing SQLite ORM functionality.

## Key Components

- **core/** - Core database operations (connection, CRUD, table sync)
- **builders/** - Query builder for complex SQL queries
- **exceptions/** - Custom exception classes (WsqliteException, etc.)
- **types/** - Pydantic to SQLite type mapping
- **cli/** - Command-line interface (wsqlite CLI command)
